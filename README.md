# DevForge CLI

DevForge is a modular CLI to quickly scaffold modern frontend applications.  
It provides an interactive setup to generate Angular or React projects with a clean, scalable structure.

## Features (MVP)

- Interactive CLI with splash screen
- Generate Angular projects
- Generate React projects (Vite or Next.js planned)
- Modular and extensible architecture for future generators

## Installation (local dev)

Clone the repo:

```bash
git clone https://github.com/mehdi-zayani/devforge.git
cd devforge
npm install
npm run build
npm link
devforge --help
```
-You Should see:
```bash
Usage: devforge [options] [command]

DevForge CLI

Options:
  -V, --version   output the version number
  -h, --help      display help for command

```
